aula 02: repositorio Local e Remoto:

aqui vamos conversar sobre GitHub:

e para explicar o que é vamos explicar o que e repositorio local e remoto

Repositorio Local: quando voce esta com um projeto voce modifica ele varias vezes
e este repositorio local seja usado para isso
onde sechama git

Git: e um gerenciador de versões que salva todas as versões de um projeto

Repositorio Remoto: então esta possue quase a mesma função, só que com a 
diferença de ser online, então se voce perder o computador ele sera salvo neste
repositorio

GitHub: e praticamente isso, onde usa o git sitado anteriormente para trabalhar
com as versões

		|> Local = Git    |
então lembre: 			  |}e quando citamos atualizações remotas dizemos
				  | (Push)
		|> Remoto = GitHub|
