PASSO 01: anatomia e regras

aqui vamos revisar o que e Media Quiries e como funciona

Basico

Media Queries e uma ferramenta que seta stylese os separa que e selecionado pelo tamanho da tela

Extrutura

ela e uma ferramenta que se coloca no link:css, no final dele 

        link rel="stylesheet" 
        href="estilos/tela.css" 
        media="formato"

Objetos da Extrutura

  • media: ele separa os estilos para dividir em diversas midiasqueres

  • screen = para tela normal, o site original

  • print = para impresão do site

  • all = todos

E para finalizar: Existe uma extrategia chamada 'Mobile First' onde vc ao usar essa ferramenta vc faz a versão do celular primeiro
