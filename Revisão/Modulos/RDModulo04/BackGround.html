BackGrounds

aqui vamos falar tudo sobre backgrounds, imagens, shorthands e outras configs

Colocando uma imagem de fundo no seu site

bem aqui n tem muito segredo: vai no style e escreve background-image: url(); em seguida dentro do url() voce coloca o indereço da imagem

Imagens que se repetem no fundo do site

aqui vamos esplicar uma coisa: quando a imagem n ocupa totalmente o site, ele se repete, então para resolver isso ou mexer com isso podemos fazer varias coisas como: background-repeat: onde voce pode configurar para repetir apenas no eixo x ou y, ou n repetir ex: background-repeat: no-repeat; n vai repetir background-repeat: repeat-x; vai repetir no eixo x background-repeat: repeat-y; vai repetir no eixo y

Configurando a posição da imagem no fundo

bem pode rolar o caso da imagem ser maior que o site, por conta disso a gente pode escolher qual parte da imagem vai ser apresentado: background-position onde aqui voce escolhe entre top, bottom, center, left ou right sendo: bottom: baixo; top: topo; center: centro; left: direita; right: esquerda; ficando assim: background-position: position;

Mudando o tamanho da imagem

agora vamos falar sobre os tamanhos: podemos colocar um tamanho especifico no background como 0px, 0em, etc... ou podemos conf' com cover, ou auto: auto: conf' automaticamente para a versão normal que seria repetir e n ocupa tudo cover: faz com que ocupe todo o site ex: background-size: auto; ou background-size: cover; obs: n da para colocar ele no shorthand, tem que colocar separado

attachmant e short-hand

bem vamos comesar com attachmant onde ele conf' como o background vai se comportar de acordo com o scrol e ele tem dois principais que são: fixed: ele seguira o usuario deixando apenas o conteudo rolando scroll: ele se repetira no eixo y para seguir o conteudo ex: background-attachment: compor'; e temos as short-hand que seria uma simplificação de todas essas styles para background. Anatomia: color > image > position position > repeat > attachment; onde para relembrar aqui o size n da para colocar aqui então ficaria assim: background: color url() position position repeat attachment;

organização vertical de caixas

aqui vamos falar sobre organizações onde podemos organizar objetos atravez do style com a tag: position; ex:

codigo

#container {
    height: 200px;
    background-color: black;
    position: relative;
}

#contido {
    height: 50px;
    width: 100px;
    position: absolute;
    background-color: yellowgreen;

    top: 40%;
    right: 45%;
}
