O que são tabelas?

tabela e uma ferramenta para a organização de dados, bem vc sabe:

Tabelas

e para fazermos as tabelas precisamos saber sua anatomia que seria:

tabela
    thead
        tr
            th
            td
        /tr
    /thead
    tbody 
        tr
            th
            td
        /tr
    /tbody
    tfoot 
        tr
            th
            td
        /tr
    /tfoot

onde: tabela = table, onde seria por onde comesa a tabela

thead, tbody, tfoot = e como formamos as tabelas, sendo a mesma coisa para um arquivo html

tr = e o que forma uma linha na tabela

td, th = são quase a mesma coisa: td: e para dados th: e para titulos

Caption e Escopos

Caption

são breves descrições que podem ficar acima dastabelas 

table
    caption
    /caption
/table

exemplo:

Caption
thead titulo
tbody conteudo
tfoot finalizações

e escopos são uma clasifiação para th's onde define de quem e o titulo, se e da coluna ou da linha

th scope=""

e no scope pode ter 2 valores, sendo eles:

col = define as colunas

row = define as linhas

colgroup = define quando o mesmo titulo possue mais de uma coluna

rowgroup = define quando um titulo possue mais de uma linha

ex:

dono dos alinhados 1 2
1 2
1 2
