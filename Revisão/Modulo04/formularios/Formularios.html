PASSO 01: anatomia e regras

aqui revisaremos como criamos um formulario e como funciona

Basico

formularios são uma ferramenta usada para coletar info do usuario onde ao ser mandado para um arquivo de programação por logica ele analisa as informações e manda um valor ou leva a um link resultante

Extrutura 

    form
        fieldset
            legend/legend
                p
                    label/label
                    input/input
                /p
        /fieldset
    /form
Objetos da Extrutura

    Form

  • action: ele e onde vc soloca o arquivo que sera mandado as info
    e a action aceita duas linguagem que são:
    php = e linguagem para informações
    javaScript = e linguagem de logica

  • method: ele escolhe qual sistema de segurança ele usara para mandar as info:

    • post: quando as informações n são tão importamtes, n são senciveis, usar quando n tiver transferenciade de arquivo, n pode passar de 3000 bites ou seja 3000 letras

    • get: usar apenas quando as informações forem censiveis, quando tiver transferencia de arquivos, e se o form for maior que 3000 bites

  • autocomplete: ele liga a função de autocompletar os inputs

    Label

  • for: ele liga a input com a label onde para ligar tem que colocar o name ou a id da input

    Input

  • type: ele mostra qual e o tipo do input

  • name: da o nome para o input onde vai ligar ao label onde sera usado em form que usara arquivos php 'form action = arquivo.php'

  • id: da a id para o input onde vai ligar ao label onde sera usado em form que usara arquivos javaScript 'form action = arquivo.javaScript'
