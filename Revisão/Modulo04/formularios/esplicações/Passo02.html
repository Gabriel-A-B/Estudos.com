Selected

select e uma ferramenta usada para colocar opções ja escolhidas para valerem aqui

Extrutura

form

    p
        label for="iEst"Estado:/label
        select name="Est" id="iEst"

        /select
    /p
/form

onde para criar suas opções podemos usar as tags: 

optgroup label="Lista"

    option value="Op" Opção /option

/optgroup

Explicações de objetos

  • optgroup = ele divide em grupos as opções

  • option = ele e a tag que cria uma opção

  • value = neste caso ele apenas e para criar siglas da opção

  • label = neste caso e só para nomear o grupo

  • selected = e uma conf que ao colocar ele vai marcar a option com ela como valor basico, isso e, ele aparecera quando o usuario n tiver escolhido um

Datalist

Datalist = e semelhante ao selected com a diferença de que nele vc pode colocar opções a mais

Extrutura

p
    label for="iDList" DList /label
    input type="text" name="DList" 
    id="iDList" list="List"

    datalist id="List"
    /datalist
/p

Explicações de objetos

  • list = ele interliga a tag do datalist com a input

TextArea

TextArea = Ele cria um bloco que permite textos compridos onde são muito usados para descrições ou comentarios

Extrutura

p
    label for="iTAre" TextArea /label

    textarea name="TAre" id="iTAre" 
    cols="30" rows="10"

    /textarea
/p

Explicações de objetos

  • cols = define a quantidade de colunas que pode ser posta

  • rows = ele define a quantidade de linhas que podem ser escrita
