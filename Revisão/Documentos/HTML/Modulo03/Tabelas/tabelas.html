PASSO06: Scope de grupos

Explicação

Lembra quando usamos os scopes 'row' e 'col' para fazer com que um 'th' represente uma coluna ou uma linha na hora do css em questão de tamanho, sendo assim, o scopes de grupo e usado quando um 'th' quepresenta mais de uma linha ou coluna

Extrutura

        table
        tr
            td----------/td
            th colspan="2" scope="colgroup"----------/th
            th colspan="2" scope="colgroup"----------/th
        /tr
        tr
            th rowspan="2" scope="rowgroup"----------/th
            td----------/td
            td----------/td
            td----------/td
            td----------/td
        /tr
     /table

Objetos

Exemplo

X V colgroup colgroup V
rowgroup > A B C D
A B C D
