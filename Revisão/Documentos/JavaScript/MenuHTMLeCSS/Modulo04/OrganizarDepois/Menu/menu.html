PASSO 01: Criação Do Menu

aqui vamos mostrar como criar menus interativos com JavaScript

e para comesar vamos primeiro pegar um icone para o menu

Icone:

primeiro precisamos pegar um icone e podemos usar o site: https://fonts.google.com/icons?icon.set=Material+Icons

apos isso vc faz a extenção para o icone que sera pego usando a internet assim

  • parte01: vc extende com a tag link onde estara nas instruções do icone abaixo dele apos o clique:

    link href="https://fonts.googleapis.com/icon?family=Material+Icons" rel="stylesheet"

  • parte02: apos isso vc incrementa o icone ao codigo com esta extrutura: 

            
    span class="material-icons" id="iMenu" 
    onclick="clickMenu()"
        menu
    /span

    Objetos:

    • span: tag usada pelo site para incrementar um icone para uso do codigo

    • class: para reconhecer como icone com apenas uma informação inportante, ele tem que estar escrito apenas "material-icons"

    • id: ele coloca uma id para linkar com o script de Java

    • onclick: conf de scripts que ativa certo objt ao ser clicada

    • menu: o icone desejado, aqui vc coloca o nome do icone desejado

Script:

apos isso usamos o Java para criar uma logica no menu para extender caso clicado e compactado caso clicado denovo

primeiramente criamos um conjunto de objetos em sequencia e compactamos ela em um unico grupo, para assim podermos colocar uma id nela, por exemplo 

    menu id="iItens"
        ul
            lia href="#"Opção001/a/li
            lia href="#"Opção002/a/li
            lia href="#"Opção003/a/li
            lia href="#"Opção004/a/li
            lia href="#"Opção005/a/li
        /ul
    /menu
em seguida, no css colocamos no "iItens" um display com valor none e criamos uma logica no script

Extrutura do Script

    script
        function clickMenu() {
            if(iItens.style.display == 'block') {
                iItens.style.display = 'none'
            }else {
                iItens.style.display = 'block'
            }
    /script

Objetos

  • function: ela cria uma logica para um objeto de script

  • if e else: cria posibilidades que analisaram a situação descrita e muda o valor de acordo com seu objetivo

  • iItens.style.display: e uma localização para o objeto que deseja mudar seu valor, onde tem como extrutura:

    idDoItem.arquivo.confQueDesejaMudarSeuValor = 'valor'
    ou
    idDoItem.arquivo.confQueDesejaMudarSeuValor == 'valor'

PASSO 02: Consertando Bug

assim que ele muda de mediaQuere ele buga e n permite ser fechado, então para consertar vamos forçar a logica para funcionar

Parte do HTML

para comesar precisamos criar um objeto de script para o body usando a ativação chamada 'onresize'

Extrutura

para fazermos isso iremos coloca-lo no body 

        body onresize="sizingTela()"

  • onresize: onde ativa quando detecta que um objeto do site mudou de tamanho, ou se o proprio site mudou de tamanho

  • sizingTela(): Nome do objeto para o script

Parte do Script

e agora vou mostrar a extrutura do script

Extrutura

colocamos isso no script 

    function sizingTela() {
        if(window.innerWidth >= '768') {
            iItens.style.display = 'inline-block'
        }else {
            iItens.style.display = 'none'
        }
    }

  • window.innerWidth: ela e uma função do jav que verifica o tamanho da tela para que assim possa ou n ser ativada caso for '>= maior' ou '<= menor'
