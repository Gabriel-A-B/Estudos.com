PASSO 02: Tipos de backgrounds

Explicação

na area de background possuimos diversos tipos de backgrounds sendo eles

Tipos de backgrounds

  • normal: background comum, ao qual vc escolhe apenas uma cor 

            body {
            background: #000000;   
        }

  • linearColor: ele muda crecentemente de uma cor para outra, de extrutura para extrutura 

            body {
            background-image: linear-gradient
            (to bottom, color1, color2);
        }

    onde vc pode mudar a direção com to top, to bottom, to left, to right

  • radialColor: ele muda crecentemente de uma cor para outra 

            body {
            background-image: radial-gradient
            (color1, color2);
        }

  • Imagens: ele usa url de imagens para coloar a imagem escolhidas como background 

            body {
            background-image: url(urlDaImagem)
        }

PASSO 03: Possition

Explicação

ele coloca para focar uma parte expecifica de uma background-image: url

Tipos de Position

  • top, bottom, left, right: eles focam extremidades da imagem para mostrar no background 

            body {
        background-image: url();
        background-possition: top ou bottom, left 
        ou right;
        }

  • center: ele foca no centro da imagem do background 

            body {
            background-image: url();
            background-possition: center;
        }

PASSO04: Repeat

Explicação

ele seta se sera repetido a imagem de fundo casso ele n ocupa toda a janela

Tipos de Repeat

  • repeat: ele repete uma imagem caso n ocupe toda a janela 

            body {
            background-image: url();
            background-repeat: repeat;
        }

  • no-repeat: ele só n repete 

            body {
            background-image: url();
            background-repeat: no-repeat;
        }

  • repeat-x e repeat-y: ele repete a imagem no eixo selecionado como X ou Y 

            body {
            background-image: url();
            background-repeat: repeatX ou repeatY;
        }

PASSO 05: Size

Explicação

ele cuida do tamanho d background onde pode cetar um tamanho em especifico, ou podemos usar as pretamanhos que tem

Tipos de Size

  • auto: ele e o que ja esta presetado onde o sistema ja escolheo o tamanho e como se comportara 

            body {
            background-image: url();
            background-size: auto;
        }

  • contain: ele faz com que o background ocupe copletamente um container como por Exemplo o main 

            body {
            background-image: url();
            background-size: contain;
        }

  • cover: ele adapta o background para que ele se expanda para suportar a imagem toda 

            body {
            background-image: url();
            background-size: auto;
        }

    • a diferença do contain e do cover e que:

    o contain: ele privilegia o conteudo

    o cover: ele privilegia a imagem

Exemplo

PASSO06: attachment

Explicação

como um background se comporta em rolagem de pagina

Tipo de Attachment

  • fixed: ele deixa o background fixo em um unico local do site 

            body {
            background: #000000;
            background-attachment: fixed;
        }

  • local: ele esta sempre seguindo o usuario 

            body {
            background: #000000;
            background-attachment: local;
        }

  • scroll: ele segue a rolagem da pagina em uma extremidade da pagina cetada 

                    body {
                    background: #000000;
                    background-attachment: scroll;
                }
