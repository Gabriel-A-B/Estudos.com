PASSO01: Basico

Explicação

tabelas são formas de organizar informações, onde tinham uma epoca em que usavam para fazer uma pagina, so que hoje em dia n se pode mais fazer isso, pois o google n recomndera-ra seu site e esta sugeito a sql inchektion

Extrutura

        table
            tr
                th----------/th
                td--------/td
            /tr
        /table

Objetos

Exemplo

Teste02 1 2 3 4 5
Teste03 1 2 3 4 5
Teste04 1 2 3 4 5
Teste05 1 2 3 4 5

PASSO04: Efeito Zebrado em tabelas

Explicação

efeito Zebrado e auto explicativo, e um metodo capaz de colocar uma background-color em um e em outro n

E tem milhares de formas para fazer isso, porem iremos anotar esssa

Extrutura

        tabela tr:nth-child(2n) {

        }

Objetos

Dado 01
Dado 02
Dado 03
Dado 04

Observações

tem como fazer com par e inpar, onde ao invez de usar numeros, podemos usar 'odd' e 'even' ou podemos usar questões matematicos assim 3n-1 ou 5n+3

Explicação

tem como colocar o abeçario fixo no topo da tela em tabelas grandes

Extrutura

        table#Cab {
            position: relative;
        }

        table#Cab > tr > thead {
            position: sticky;
            top: -1px;
