PASSO06: Utilizando css em scopes

Explicação

tem um geito de editar uma coluna inteira usando as socpes

Extrutura

    style

        col.primeiroCol {
            background-color: cadetblue;
        }

        col.segundoCol {
            background-color: thistle;
        }

    /style

    table
        colgroup
            col class="primeiroCol"
            col class="segundoCol"
        /colgroup

        tr
            th scope="col" class="primeiroCol"
                ----------
            /th
            th scope="col" class="segundoCol"
                ----------
            /th
        /tr
        tr
            td----------/td
            td----------/td
        /tr
    /table

Objetos

Exemplo

Grupo01 Grupo02
1 2
1 2

PASSO07: Tabelas Responsivas

Explicação

normalmente as tabelas ja vem com responsividade, porem ate um certo ponto, onde acaba criando uma rolagem lateral, aqui te insinarei a resolver isso

Extrutura

        style
            var#Container {
                overflow-x: auto;
            }
        /style

        var id="Container"
            table
                tabela muito longa
            /table
        /var

Objetos

overflow-x: ele cria uma rolagem lateral (no eixo x) dentro da tabela caso ela n conseguir diminuir mais

Exemplo
