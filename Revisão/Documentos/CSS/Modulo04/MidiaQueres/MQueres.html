PASSO 01: anatomia e regras

aqui vamos revisar o que e Media Quiries e como funciona

Basico

Media Queries e uma ferramenta que seta stylese os separa que e selecionado pelo tamanho da tela

Extrutura

ela e uma ferramenta que se coloca no link:css, no final dele 

        link rel="stylesheet" 
        href="estilos/tela.css" 
        media="formato"

Objetos da Extrutura

  • media: ele separa os estilos para dividir em diversas midiasqueres

  • screen = para tela normal, o site original

  • print = para impresão do site

  • all = todos

E para finalizar: Existe uma extrategia chamada 'Mobile First' onde vc ao usar essa ferramenta vc faz a versão do celular primeiro

PASSO 02: Extrutura em Css

Basico:

as Media Queres tbm podem ser feitas em arquivos css onde podem ser divididas em diversas Medias

Extrutura 

@media screen and (orientation: Orientação) {}

Objetos da Extrutura

  • @media: ela separa os estilos para dividir em diersas midiaQueres

  • screen: e o formato da midia quere

  • orientation: ele e a config para o tamanho da tela como width e height com valores pre-definidos

valores

  • midia:

    • screen = e para telas em geral

    • print = e para a janela de impresão do site

  • orientation:

    • portrait = para celulares em pé

    • landscape = para celulares deitados

PASSO 03: Tamanhos mais expecificos

Basico:

possue uma certa lista de tamanhos de telo onde usamos para cetar certos tipos de tamanho de telas e midias queres usados

Extrutura: 

@media  screen and (min-width: 600px) and (max-width: 768px) {}

Objetos da Extrutura:

  • min-width: ele seta o minimo do tamanho de tela para mudar

  • max-width: ele seta o maximo da tela para mudar para proxima midia quere
